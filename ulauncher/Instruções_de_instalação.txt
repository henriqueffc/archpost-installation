Inicialize o Ulauncher

# Atalho criado para o Ulauncher no script de instalação (número 2) "Tecla Super + \"

# Marque a opção Launch at login

# Para o ícone do Ulauncher aparecer na top bar é necessário instalar a extensão do Gnome "AppIndicator and KStatusNotifierItem Support". 
# Recomendo utilizar o programa Extension Manager - https://flathub.org/apps/details/com.mattjakeman.ExtensionManager (instalado pelo scrpit de instalação número 3) para instalar a extensão. 
# Caso você não queira que o ícone apareça na top bar desabilite essa opção nas configurações do Ulauncher.


# Instalação do tema

git clone https://github.com/dracula/ulauncher.git ~/.config/ulauncher/user-themes/dracula-ulauncher

git clone https://github.com/tom-james-watson/ulauncher-popdark.git ~/.config/ulauncher/user-themes/ulauncher-popdark

# Altere o tema nas opções do aplicativo.


# Shortcuts e Logos

mv ~/.config/ulauncher/shortcuts.json ~/.config/ulauncher/shortcuts.json.bak
mv shortcuts.json ~/.config/ulauncher/
tar -Jxxvf imagens.tar.xz -C ~/Imagens

# A instrução acima copia as configurações dos shortcuts salvas anteriormente e descompacta os logos desses shortcuts na pasta Imagens na Home.
# Não altere o nome da pasta (Logo Ulauncher).
# Reinicie o Ulauncher após essa configuração para habilitar os shortcuts.
