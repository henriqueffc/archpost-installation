# Coisas para fazer depois de concluída a instalação com os scripts:

1 - Habilitar o cache de escrita no programa Discos;


2 - Configurar no Flatseal as variáveis de ambiente para os aplicativos que estejam com problemas de tema.
GTK - GTK_THEME=Adwaita-dark ou QT_STYLE_OVERRIDE=adwaita-dark 


3 - Verifique o policy.xml do ImageMagick;


4 - Atalho criado para o Ulauncher no script de instalação (número 3) "Tecla Super + \"

Para o ícone do Ulauncher aparecer na top bar é necessário instalar a extensão do Gnome "AppIndicator and KStatusNotifierItem Support". 

Recomendo utilizar o programa Extension Manager - https://flathub.org/apps/details/com.mattjakeman.ExtensionManager (instalado pelo scrpit de instalação número 4) para instalar a extensão. 

Caso você não queira que o ícone apareça na top bar não habilite essa opção nas configurações do Ulauncher.

Página para pesquisar extensões: https://ext.ulauncher.io/

Página para pesquisar os temas: https://gist.github.com/gornostal/02a232e6e560da7946c053555ced6cce

Instruções para a instalação dos temas: https://docs.ulauncher.io/en/latest/themes/themes.html


5- Remover o Powerprofilesctl (Gnome - Perfil de energia)

Não alterou a velocidade do sistema no meu notebook - use o cpupower instalado pelo script 3-pacotes.sh para essa função.

sudo pacman -R power-profiles-daemon

systemctl disable power-profiles-daemon.service (desabilitou o serviço momentaneamente. foi preciso desinstalar do sistema)

https://wiki.archlinux.org/title/GNOME#Power_modes
https://wiki.archlinux.org/title/CPU_frequency_scaling
https://gitlab.freedesktop.org/hadess/power-profiles-daemon


6 - GNOME
Para inserir pastas no tracking insira as opções em Configurações - Pesquisa - Pesquisar localizações - Adicionar localização


8 - Distrobox

Atalho para o terminal

No terminal acesse Preferências - Criar um novo perfil - Comando - na opção Comando personalizado (marque a opção Executar um comando personalizado ...) digite distrobox enter ubuntu  
Mude as cores do perfil para especificar a diferença de uso do terminal (padrão ou distrobox).

Em configurações - Teclado - Atalhos de teclado

Crie um atalho personalizado.

Nome : terminal ubuntu
Comando : gnome-terminal --profile=ubuntu
Atalho : Super + U
