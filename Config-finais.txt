# Coisas para fazer depois de concluída a instalação com os scripts:

1 - Habilitar o cache de escrita no programa Discos;


2 - Configurar no Flatseal as variáveis de ambiente para os aplicativos que estejam com problemas de tema. Exemplo:
GTK_THEME=Adwaita-dark ou QT_STYLE_OVERRIDE=adwaita-dark 


3 - Verifique o policy.xml do ImageMagick;


4- Remover o Powerprofilesctl (Gnome - Perfil de energia)

Não alterou a velocidade do sistema no meu notebook - use o cpupower instalado pelo script 3-pacotes.sh para essa função.

sudo pacman -R power-profiles-daemon

systemctl disable power-profiles-daemon.service (desabilitou o serviço momentaneamente. foi preciso desinstalar do sistema)

https://wiki.archlinux.org/title/GNOME#Power_modes
https://wiki.archlinux.org/title/CPU_frequency_scaling
https://gitlab.freedesktop.org/hadess/power-profiles-daemon


5 - Configurar rkhunter, Fail2ban - porta SSH, SSH e Lynis (https://github.com/CISOfy/Lynis)
